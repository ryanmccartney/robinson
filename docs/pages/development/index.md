# Development

-   `git clone https://github.com/ryanmccartney/robinson`
-   `cd ./backend && npm i`
-   `cd ./frontend && npm i`
-   `docker compose up -d`

Find the development frontend on `http://localhost:3000` and self-documenting API backend at `http://localhost:3100/api`
